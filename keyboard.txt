6Ô∏è‚É£ Prepare the Bill (show total)

MongoDB doesn‚Äôt calculate totals automatically,
but you can use the aggregation pipeline to compute it:

db.Orders.aggregate([
  { $match: { order_id: 101 } },
  { $unwind: "$items" },
  {
    $group: {
      _id: "$order_id",
      customer_name: { $first: "$customer_name" },
      total_amount: { $sum: "$items.price" },
      items: { $push: "$items" }
    }
  }
])
